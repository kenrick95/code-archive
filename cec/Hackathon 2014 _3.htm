Problem Description

Given a string, find the longest substring with the same characters. If multiple longest substrings found, return the first one.

Input Format

Each line contains a string. Each character in the string is in lowercase format.

Output Format

The start index and the longest substring with the same characters itself.

Constraints:

1 <= S <= 10^5 (S is the size of the string)

Sample Input File

efcaadddegmmmmmmc
tyyqeedbcbszzzwrrtyufgd

Sample Output File

10 mmmmmm
11 zzz
