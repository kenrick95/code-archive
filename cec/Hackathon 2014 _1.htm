Problem Description

Regina and Gretchen go with their friend Cady to indulge in a shopping spree. However Cady only has a limited amount of money. Her friends want her to buy as many clothes as she can. They suggest N items but Cady only has K dollars.

Help Cady buy as many clothes as possible.

Input

For each problem, there are two lines. The first line of each problem will contain two inputs N and K, followed by a line containing N integers each representing the price of a clothing item.

Output

Maximum number of clothes Cady can buy.

Constraints

1<=N<=10^4

1<=K<=10^8

1<=price of any clothing item<=10^8

Sample Input File

8 500
30 120 50 110 200 1000 100 150
5 1000
75 60 1000 160 500

Sample Output File

5
4
