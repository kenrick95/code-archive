Problem Description

There are N cities in the province of Eastern Cape. A cargo company helps clients deliver cargos among these N cities. There may be one transportation link or none between any given pair of cities. The transportation fee consists of two parts:

  • The cost of the transportation on the route between these cities, and
  • A tax which will be charged whenever cargo passes through a city that is not the source or destination city

Please help find the route between two given cities which incurs the lowest cost.

Input

The first input is N, number of cities. N = 0 indicates the end of input.

The data about the cost of each path, tax charged by each city, and the source and destination cities are given in the input, which is of the following form:

a11 a12 ... a1N
a21 a22 ... a2N
...............
aN1 aN2 ... aNN
b1  b2  ... bN

c d
e f
...
g h

where aij is the transport cost from city i to city j, aij = -1 indicates there is no direct path between city i and city j. bi represents the tax of passing through city i. And the cargo is to be delivered from city c to city d, city e to city f, ..., and g = h = -1. Your program must output the sequence of cities to be passed through and the total cost in the following form:

Output

From c to d :
Path: c-->c1-->......-->ck-->d
Total cost : ......
......

From e to f :
Path: e-->e1-->..........-->ek-->f
Total cost : ......

Note: if there are more than one paths incurring the same lowest cost, output the lexically smallest one. Print a blank line after each test case.

Sample Input File

5
0 3 22 -1 4
3 0 5 -1 -1
22 5 0 9 20
-1 -1 9 0 4
4 -1 20 4 0
5 17 8 3 1
1 3
3 5
2 4
-1 -1
0

Sample Output File

From 1 to 3:
Path: 1-->5-->4-->3
Total cost: 21

From 3 to 5:
Path: 3-->4-->5
Total cost: 16

From 2 to 4:
Path: 2-->1-->5-->4
Total cost: 17
